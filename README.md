## 什么是搬瓦工 VPS 的 IP 更换？

搬瓦工（BandwagonHost）是 IT7 Networks 旗下的 VPS 服务商，每台 VPS 出厂配一个独立 IPv4 地址。这个 IP 地址是固定的，用户无法自行随意更换。

当 VPS 的 IP 被国内防火长城封锁后，会出现以下现象：国内 SSH 连不上、Ping 不通，但 VPS 本身在正常运行（海外节点能访问）。这时候就需要"更换 IP 地址"。

目前搬瓦工 IP 更换有三条路：**付费向官方申请换 IP**、**切换机房（仅限支持多机房的套餐）**、**直接买新的 VPS**。没有其他捷径。网上流传的免费换 IP 服务是旧信息——那个功能早在 2019 年初就已下线，不再支持。

## 第一步：先确认 IP 是否真的被封了

很多人连不上第一反应就是"IP 被封了"，但其实端口问题、本地网络、VPS 宕机也会造成同样的症状。在花钱换 IP 之前，先用搬瓦工自带的检测工具确认一下。

**操作步骤如下：**

1. 登录搬瓦工账户，进入 Services → My Services
2. 找到对应 VPS，点击 **KiwiVM Control Panel** 进入控制面板
3. 在控制面板中，导航到 IP 检测功能（或在浏览器地址栏手动修改 URL，把路径改为 `main-exec.php?mode=blacklistcheck`，其中 VMID 替换成你自己的 VPS ID）
4. 点击 **Test Main IP** 按钮

结果有两种：
- `IP NOT BLOCKED`：IP 正常，问题不在 IP，去排查别的原因
- `IP BLOCKED`：IP 确认被封，这才需要换

说实话，这一步很多人跳过了，直接去申请换 IP，结果换完发现还是连不上——因为根本不是 IP 的问题。

IP 被封确认之后，换 IP 的流程不复杂，走官方页面申请就行。

5. 支付后系统自动生成工单，等待处理

**费用：$8.79 / 次**（以实际页面显示为准，可能有小幅调整）

换完之后记得两件事：一是在 KiwiVM 面板查看新 IP；二是如果你的 VPS 上跑了网站，域名 DNS 解析要及时指向新 IP，不然网站还是打不开。

## 有没有不花钱换 IP 的方法？

有，但只适合特定套餐的用户。

**方法一：切换机房**

买的是 CN2 GIA-E 套餐的用户，这个套餐支持在 12 个以上机房之间自由切换。每次切换机房，IPv4 地址就会变。相当于间接实现了"免费换 IP"的效果。

这是目前性价比最高的绕弯方式——你不是在付费换 IP，而是在换一个新机房，IP 附赠。如果上一个机房的 IP 被封了，切到另一个机房继续用，就这么简单。

不过要注意一个限制：**如果当前 IP 已经被封，切换机房的功能会被锁住**，必须先付费换一个可用 IP，才能操作机房迁移。所以最好的状态是 IP 还没封的时候就提前切走。

搬瓦工在 2025 年底推出了 DC5 SLA 套餐，内置**每 14 天可免费换一次 IP** 的政策，且 IP 被封也在免费换的范围内。这是目前搬瓦工体系里唯一有这个待遇的常规套餐。

## 常见问题 FAQ

**Q：搬瓦工 IP 被封后多久能自然解封？**

A：不固定。有人等了一个月自己好了，有人等了大半年还没动静。如果有正常使用需求，等不起，付费换或者切机房是更可控的方案。

**Q：换 IP 之后原来的 IP 会怎样？**

A：原 IP 回到地址池里，官方会做可用性验证再分配给其他用户。你拿到的新 IP，搬瓦工保证在全球各地均可访问。

**Q：KVM 入门套餐能切机房吗？**

A：不一定，部分 KVM 套餐不支持机房迁移。购买前在官网套餐页面确认可选机房范围。如果有机房切换需求，优先选 CN2 GIA-E 套餐，支持 12 个以上机房自由迁移。

**Q：搬瓦工购买时需要填写邀请码吗？**

A：不需要。直接通过官网正常购买即可，不存在必须填写邀请码才能享受优惠的情况。下单时可以在结算页面输入官方促销码享受折扣，具体以官网当前活动页面显示为准。

**Q：付费换 IP 之后，能保证新 IP 不被封吗？**

A：不能。搬瓦工只保证新 IP 在全球各地当下可访问，不能保证未来不会被封。根本上还是看使用方式。
